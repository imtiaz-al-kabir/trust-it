# Trust-IT (BD) Ltd - Landing Page

## Project Overview
This project is a responsive and visually appealing landing page for **Trust-IT (BD) Ltd**, a fictional IT company. It is built as part of a junior developer assessment to demonstrate skills in clean UI design, semantic HTML, responsive layouts, and Vanilla JavaScript interactivity.

## Technologies Used
- **HTML5**: For semantic and accessible structure.
- **Tailwind CSS (CDN)**: For modern, utility-first styling and premium aesthetics.
- **Vanilla JavaScript**: For all interactivity including Dark Mode, form validation, and mobile navigation.
- **Google Fonts**: 'Inter' typeface for professional typography.

## Instructions to Run the Project
1. **Download/Extract**: Ensure all project files and folders (`assets/`, `js/`, `index.html`) are in the same directory.
2. **Open index.html**: Double-click the `index.html` file in your preferred web browser.
3. **Internet Connection**: Ensure you have an active internet connection to load Tailwind CSS and Google Fonts via CDN.
4. **No Build Required**: This project runs directly in the browser without needing `npm install` or any build steps.
